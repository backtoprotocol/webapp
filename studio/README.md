# Back to Protocol Studio

1. The Studio is connected to the Back to Protocol project (`lcmmmi2e`) and the `production` dataset.
2. Run `npm run dev` from this folder to open the authoring Studio.
3. In Sanity Manage, add `http://localhost:3000` to CORS Origins before using published content in the web app.

Create an **Author** first, then create an **Article**. Every article requires a pillar, author, publish date, reading time, and slug. The site automatically creates related coverage from the same pillar.
